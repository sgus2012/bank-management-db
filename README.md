# Bank Management Database

Учебный проект базы данных банка на PostgreSQL.

## Описание проекта

Проект представляет собой структуру базы данных банковской системы.

База хранит информацию о:

- клиентах банка;
- счетах;
- банковских картах;
- транзакциях;
- кредитах;
- платежах по кредитам;
- сотрудниках банка.

## Используемые технологии

- PostgreSQL
- SQL
